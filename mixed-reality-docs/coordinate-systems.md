## <a name="spatial-coordinate-systems"></a><span data-ttu-id="88e84-189">空間座標系</span><span class="sxs-lookup"><span data-stu-id="88e84-189">Spatial coordinate systems</span></span>

<span data-ttu-id="88e84-190">すべての 3D グラフィックス アプリケーションを使用して、[デカルト座標系](https://docs.microsoft.com/windows/uwp/graphics-concepts/coordinate-systems)をレンダリングする仮想世界のオブジェクトの向きと位置を判断します。</span><span class="sxs-lookup"><span data-stu-id="88e84-190">All 3D graphics applications use [Cartesian coordinate systems](https://docs.microsoft.com/windows/uwp/graphics-concepts/coordinate-systems) to reason about the positions and orientations of objects in the virtual worlds they render.</span></span> <span data-ttu-id="88e84-191">このような座標システムは、3 つの垂直軸に沿って配置するオブジェクトの位置を確立します。 X、Y、および Z 軸。</span><span class="sxs-lookup"><span data-stu-id="88e84-191">Such coordinate systems establish 3 perpendicular axes along which to position objects: an X, Y, and Z axis.</span></span>

<span data-ttu-id="88e84-192">[複合現実](mixed-reality.md)、仮想および物理の両方の座標システムに関する理由は、アプリ。</span><span class="sxs-lookup"><span data-stu-id="88e84-192">In [mixed reality](mixed-reality.md), your apps will reason about both virtual and physical coordinate systems.</span></span> <span data-ttu-id="88e84-193">Windows の呼び出しを現実の世界で本当の意味を持つ、座標系、**空間座標系**します。</span><span class="sxs-lookup"><span data-stu-id="88e84-193">Windows calls a coordinate system that has real meaning in the physical world a **spatial coordinate system**.</span></span>
