<span data-ttu-id="bd5a4-181">HoloLens 2 では手を使用して、境界ボックスを各 3D オブジェクトに適用することで 3D hologramphic オブジェクトを直接操作するユーザーが有効化します。</span><span class="sxs-lookup"><span data-stu-id="bd5a4-181">In HoloLens 2, users are enabled to use their hands to direct manipulate 3D hologramphic objects by applying a bounding box to each 3D object.</span></span> <span data-ttu-id="bd5a4-182">境界ボックスは、その近接シェーダーを通じてより優れた深さ perception を提供します。</span><span class="sxs-lookup"><span data-stu-id="bd5a4-182">The bounding box provides better depth perception through its proximity shader.</span></span> <span data-ttu-id="bd5a4-183">境界のボックスでは、3 D オブジェクトの操作の 2 つのデザイン方法があります。</span><span class="sxs-lookup"><span data-stu-id="bd5a4-183">With the bounding box, there are two design approaches for 3D object manipulation:</span></span>      
### <a name="affordance-based-manipulation"></a><span data-ttu-id="bd5a4-184">操作をアフォー ダンスに基づいています。</span><span class="sxs-lookup"><span data-stu-id="bd5a4-184">Affordance based manipulation:</span></span>

### <a name="non-affordance-based-manipulation"></a><span data-ttu-id="bd5a4-189">操作を非アフォー ダンスに基づいています。</span><span class="sxs-lookup"><span data-stu-id="bd5a4-189">Non-affordance based manipulation:</span></span>

## <a name="instinctual-gestures"></a><span data-ttu-id="bd5a4-196">Instinctual ジェスチャ</span><span class="sxs-lookup"><span data-stu-id="bd5a4-196">Instinctual gestures</span></span>
<span data-ttu-id="bd5a4-197">HoloLens とは異なり (第 1 世代) いくつかの定義済みの授業ユーザー ジェスチャの「Bloom、HoloLens 2 でのエア タップなど、任意のシンボリック ジェスチャを記憶するユーザー メッセージを表示しません。</span><span class="sxs-lookup"><span data-stu-id="bd5a4-197">Unlike HoloLens (1st gen), teaching users a couple predefined gestures, such as Bloom and Air Tap, in HoloLens 2, we don't ask users to memorize any symbolic gesture.</span></span> <span data-ttu-id="bd5a4-198">ユーザーがホログラムと内容を操作する必要があるすべてのジェスチャは instinctual です。</span><span class="sxs-lookup"><span data-stu-id="bd5a4-198">All gestures that users need for interacting with holograms and contents are instinctual.</span></span> <span data-ttu-id="bd5a4-199">Instinctual ジェスチャを実現する方法は、UI アフォーのデザインによってジェスチャを実行するユーザーをガイドです。</span><span class="sxs-lookup"><span data-stu-id="bd5a4-199">The way to achieve instinctual gesture is to guide users to perform gestures through the design of UI affordances.</span></span> <span data-ttu-id="bd5a4-200">などの場合は、オブジェクトのグラブまたは 2 本の指のピンチでのコントロール ポイントをユーザーにすることをお勧め、オブジェクトまたはコントロール ポイントは小にします。</span><span class="sxs-lookup"><span data-stu-id="bd5a4-200">For example, if we encourage users to grab an object or a control point with two finger pinch, the object or the control point should be small.</span></span> <span data-ttu-id="bd5a4-201">ユーザーが 5 本の指グラブを実行する場合、オブジェクトまたはコントロール ポイントが比較的大きななければなりません。</span><span class="sxs-lookup"><span data-stu-id="bd5a4-201">If we would like users to perform five finger grab, the object or the control point should be relatively big.</span></span> <span data-ttu-id="bd5a4-202">ボタンと同様に、小さなボタンはユーザーを制限する大きなボタンをユーザーの手のひらで押すお勧めします中に 1 本の指を押します。</span><span class="sxs-lookup"><span data-stu-id="bd5a4-202">Similar to buttons, a tiny button would limit users to press it with a single finger, while a huge button would encourage users to press it with their palms.</span></span>

<span data-ttu-id="bd5a4-209">ホログラムを誤ってトリガーすることがなくもう手を任意の場所を移動できない場合は、ストレスも簡単になることができますが、意図したとおりに動作する場合、直接操作を魔法のようなと思われることができます。</span><span class="sxs-lookup"><span data-stu-id="bd5a4-209">Direct manipulation can feel magical if it works as intended, but can also quickly become frustrating if you can’t move your hand anywhere anymore without unintentionally triggering a hologram.</span></span>
