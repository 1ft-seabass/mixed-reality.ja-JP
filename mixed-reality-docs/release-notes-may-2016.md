## <a name="major-fixes"></a>大規模な修正

低速または誤って検出された、スペースがあります、ホログラム スペースでの問題を修正しました。 シャット ダウン中に、シェルを起動してみてくださいし続けることが、シャット ダウンの問題を修正しました。

問題を修正しました
* XAML アプリケーションを再開する機能をブロックします。
* 黒い画面といくつか、クラッシュのままには、ギザギザ行にします。
* スクロールはも引き続き使用間違った方向に一部のアプリを使用する場合します。
* 電源を引き続きデバイスであったと Led にはオフの状態がある可能性があります。
* スタンバイから再開した後は、WiFi をオフの取得でした。
* Xbox の id プロバイダーでは、ゲーマータグ セットアップを提供して、ループし、状態を取得します。
* ダウンロードしたファイルを OneDrive のファイル ピッカーで選択すると場合によっては、シェルがクラッシュします。
* キーを押すと、リンクを保持している場合がありますが壊れている、新しいタブを開いたし、ショートカット メニューが開きます。
* windows デバイスのポータルが 80 に 50 から IPD 調整をことができませんでした。

写真の問題を修正しました、
* イメージを表示、場合によっては回転により EXIF の orientation プロパティは無視されます。
* ピン留めされた写真の起動中にクラッシュすることでした。
* ビデオは最後に一時停止している場所から続行するのではなく一時停止後に再開されます。
* 再生中には、共有される場合、共有のビデオの再生を防ぐでした。
* 混合の実際のキャプチャの記録が 0.5-1 で始まるはオーディオのみのフィードの 2 つ目です。
* [同期] ボタンは、OneDrive の初期同期中に表示されなくなります。

設定で問題を修正しました、
* 環境が変更されたときに、更新が必要でした。
* ' Enter' キーボードにはいくつかのダイアログ ボックスで [次へ] をクリックすると動作しません。
* clicker が失敗した場合のペアリングを把握することでした。
* WiFi の切断に応答しなくなるし、接続が可能性があります。

Cortana を使って問題を修正しました、
* リッスンしている UI を表示するスタックでした。
* 確認する「Hey Cortana, 何できますとでも言う」かもしれませんではなく、はい/いいえをアプリを終了する要求と答えた場合は、排他モードからアプリに残ってしまうとします。
* スリープ状態にし、再開に進むには Cortana に質問する場合、UI をリッスンしている Cortana は正常に再開しません。
* クエリ「どのようなネットワークに接続しますか?」 "Am は接続されているか。" 接続されていないと、最初のネットワーク プロファイルが戻ったときに失敗します。
* UI では、「リッスン中」にフリーズしましたが、アプリの終了時にすぐに開始した音声認識をもう一度実行します。
* 場所 Cortana アプリからサインアウトするでしょうできますサインオンことに、再起動まで。
* Mixed Reality キャプチャ UI がアクティブなときに起動ことはできません。

Visual Studio での問題を修正しました、
* バック グラウンド タスクのデバッグが機能しませんでした。
* グラフィックス デバッガーでフレーム分析は機能しませんでした。
