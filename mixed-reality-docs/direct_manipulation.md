### <a name="non-affordance-based-manipulation"></a><span data-ttu-id="012d6-190">非アフォー ダンス ベースの操作</span><span class="sxs-lookup"><span data-stu-id="012d6-190">Non-affordance based manipulation</span></span>

<span data-ttu-id="012d6-191">このメカニズムで境界ボックスにアフォー ダンスはアタッチされません。</span><span class="sxs-lookup"><span data-stu-id="012d6-191">In this mechanism, no affordance is attached to the bounding box.</span></span> <span data-ttu-id="012d6-192">ユーザーは、のみ、境界ボックスを表示し、直接やり取りすることができます。</span><span class="sxs-lookup"><span data-stu-id="012d6-192">Users can only reveal the bounding box, then directly interact with it.</span></span> <span data-ttu-id="012d6-193">境界ボックスは、1 つの手でグラブされる場合平行移動とオブジェクトの回転、モーション センサーとして手のアイコンの向きに関連付けられます。</span><span class="sxs-lookup"><span data-stu-id="012d6-193">If the bounding box is grabbed with one hand, the translation and rotation of the object are associated to motion and orientation of the hand.</span></span> <span data-ttu-id="012d6-194">オブジェクトは、2 つの手をグラブ ユーザーで変換、スケールおよび回転させる 2 つのハンドの動作を相対に従ってことができます。</span><span class="sxs-lookup"><span data-stu-id="012d6-194">When the object is grabbed with two hands, users can translate, scale and rotate it according to relative motions of two hands.</span></span>
