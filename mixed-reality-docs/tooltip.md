# <a name="tooltip"></a>ヒント

![ヒント](images/UX/UX_Hero_Tooltip.jpg)

ツールヒントは通常、オブジェクトの詳細な検査時にヒントや追加情報を伝えるために使用されます。 ツールヒントを使用すると、物理環境内のオブジェクトに注釈を付けることができます。
